Pandas的安装与导入#

在终端输入以下指令安装 pandas 库

pip install pandas

如果是使用 vscode 进行开发, 你可能需要额外安装 openpyxl 库:

pip install openpyxl

安装完成后在代码中导入:

import pandas as pd

Pandas 读取数据的方式#

Pandas 主要通过以下三种方式从不同来源读取数据:

  1. pd.read_csv()
  2. pd.read_excel()
  3. pd.read_sql()

Pandas 的数据类型#

Pandas 的数据类型有 DataFrame 和 Series 一般来说, 查询单行或者单列的数据, 返回的结果为 Series 查询多行多列的数据, 返回的结果为 DataFrame

Pandas 查询数据的方式#

Pandas 查询数据的 5 种方式: 1. df.loc 2. df.iloc 3. df.where 4. df.query. 在这几种方法中, df.loc 可以兼顾查询和覆写的功能, 具体使用方法如下:

df = pd.read_csv(your_csv_path)
df.loc[:, "[column1]"] = df["column1"].str.replace("origin_str_part", "target_str_part")

上面这段代码的效果是, 查询 column1 这一整列(: 表示查询所有行), 同时将该行的 origin_str_part 全部替换为 target_str_part.

你可以通过指定 index 和 column 来查询某一行或者某一列的数据:

# 查询 index=5 的 column1, column2 下的数据, 返回 `pandas.series`
series_query = df.loc[5, ["column1", "column2"]]
# 查询 index=1和5 的 column1, column2 下的数据, 返回的是 `pandas.dataframe`
dataframe_query = df.loc[[1,5], ["column1", "column2"]]

通过切片的方式进行查询#

你也可以通过类似 Python 中切片的方式来进行取段的操作:

Pandas 的切片与 Python中的切片有些许差别, 主要是不去尾, 也就是 index=5和 "column" 也在数据查询的范围内

df.loc[[1:5], ["column1": "column5"]]

使用条件表达式查询#

loc 方法还可以与条件表达式相结合进行查询, 具体使用方法如下:

df.loc[df["column"]<10, :]

上述代码实现的是返回 column 数值小于 10 的列表.

df["column"]<10 返回的是所有索引和是否符合条件查询的结果 true or false

你会看到以下类似的结果:

index1  false
index2  false
...
index10 true
index11 true
...

你还可以通过 & 和 | 关键字进行组合查询.
