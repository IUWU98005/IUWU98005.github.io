简介#

本项目展示如何使用 Python 构建一个简单的文件系统助手程序。该助手可以理解用户的文本输入，调用指定工具函数操作文件，并通过 Gemini 模型进行任务处理。

demo地址：https://github.com/IUWU98005/mai-agent

项目结构#

.
├── __init__.py
├── main.py          # 程序入口
├── tools.py         # 工具函数模块
└── cases/           # 示例文件目录
    ├── a.go
    ├── b.py
    ├── c.java
    └── d.c

环境准备#

安装依赖：

pip install pydantic-ai python-dotenv

配置环境变量：在 .env 文件中添加 Gemini API 密钥：

GEMINI_API_KEY=your_api_key_here

关键模块说明#

  1. 工具函数（tools.py）

    定义了程序可用的基础文件操作：

    # filepath: tools.py

def read_file(name: str) -> str:
    """读取指定文件的内容"""

def list_files() -> list[str]:
    """列出目录中的所有文件"""

def rename_file(cur_name: str, new_name: str) -> str:
    """重命名文件"""

  2. 助手初始化（main.py）

    from pydantic_ai.agent import Agent
from pydantic_ai.models.gemini import GeminiModel
import tools

# 初始化模型
model = GeminiModel("gemini-2.0-flash")

# 创建助手实例
agent = Agent(
    model=model,
    system_prompt="You are an experienced programmer.",
    tools=[tools.read_file, tools.list_files, tools.rename_file],
)

  3. 主程序逻辑

    def main() -> None:
    history = []
    while True:
        user_input = input("Input: ")
        resp = agent.run_sync(user_input, message_history=history)
        history = list(resp.all_messages())
        print(resp.output)
