安装 Django#

pip install django

Django 常用命令#

使用语法：django-admin [options]

  • startproject <project_name> # 创建项目
  • startapp # 创建一个 django 应用
  • check # 校验项目完整性
  • runserver # 运行项目 e.g. python manage.py runserver
  • shell # 进入 django shell
  • test # 运行测试

数据库相关：

  • makemigrations # 生成数据库迁移文件
  • migrate # 执行上条命令创建的迁移文件
  • dumpdata # 导出数据库数据
  • loaddata # 导入数据库数据

Django 项目（Project）目录结构介绍#

.
├── manage.py
├── project_name
│   ├── __init__.py
│   ├── settings.py  # 项目配置文件
│   ├── urls.py      # 项目路由文件
│   ├── wsgi.py      # 项目 wsgi 配置文件

urls.py#

from django.urls import path
from . import views

urlpatterns = [
    path('admin/', admin.site.urls),
    path('blog/', include('blog.urls')),
    path('', include('blog.urls')),  # 首页
]

Django 应用（app）目录结构介绍#

以下是一个典型的 Django 应用目录结构：

.
├── __init__.py
├── admin.py
├── apps.py
├── migrations  # 数据库迁移文件目录
│   └── __init__.py  # migrate 命令会在该目录下生成迁移文件 e.g. `0001_initial.py`
├── models.py
├── tests.py
└── views.py

views.py#

在该文件下实现你的视图函数逻辑

from django.shortcuts import render
from django.http import HttpResponse

def index(request):
    return HttpResponse("Hello, world. You're at the blog index.")

def detail(request, article_id):
    return HttpResponse("You're looking at article %s." % article_id)

models.py#

在该文件下定义你的数据库模型

from django.db import models

class Article(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()
    pub_date = models.DateTimeField('date published')

urls.py#

在该文件下定义你的路由

from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
    path('detail/<int:article_id>/', views.detail, name='detail'),
]

使用模板系统渲染页面#

Django 模板系统的基本语法#

  1. 变量标签

    {{ [argument] }}

  2. for 循环标签

    {% for [item] in [items] %}
    [html code] 
{% endfor %}

  3. if-else 判断标签

    {% if [condition] %}
[html code]
{% else %}
[html code]
{% endif %}

  4. url 标签

    <!-- 这段代码将访问 router/arg 路径 -->
<a href = "{% url '[router]' [arg]}"></a>

使用实例#

{% if next_article %}
<li class="page-item"><a class="page-link" href="{% url 'detail' next_article.article_id %}">下一篇：{{ next_article.title }} <i class="fas fa-chevron-right ms-2"></i></a></li>
{% else %}
<li class="page-item disabled"><a class="page-link" href="#">已是最后一篇 <i class="fas fa-ban ms-2"></i></a></li>
{% endif %}
