Python Unittest 使用方法速查

August 12, 2025 · 1 min
Table of Contents
  • 待测试代码（vector.py）
  • 测试代码（test_vector.py）
  • 运行测试

首先将项目的目录结构调整为如下：

.
├── vector  # 待测试的代码
│   ├── __init__.py
│   └── vector.py
├── tests  # 测试文件夹
│   ├── __init__.py
│   └── test_vector.py
└── main.py

待测试代码（vector.py）#

class Vector:
    def __init__(self, x, y):
        if isinstance(x, (int, float)) and isinstance(y, (int, float)):
            self.x = x
            self.y = y
        else:
            raise ValueError("x and y must be numbers")
    
    def add(self, other):
        return Vector(self.x + other.x, self.y + other.y)

测试代码（test_vector.py）#

import unittest
import sys
from vector import Vector

class TestVector(unittest.TestCase):
    # 每次运行 unittest 的开始与结束打印信息
    @classmethod
    def setUpClass(cls):
        print("start")

    @classmethod
    def tearDownClass(cls):
        print("end")

    def test_init(self):
        v = Vector(1, 2)
        
        self.assertEqual(v.x, 1)
        self.assertEqual(v.y, 2)
        # self.assertTrue(v.x == 0)

        # 测试 __init__ 的 raise error 是否正常工作
        with self.assertRaises(ValueError):
            Vector("a", 2)
    
    # 满足条件下跳过该测试，并提示 msg
    @unittest.skipIf(sys.platform == "win32", "skip on windows")
    def test_platform(self):
        ...

assertEqual() 在不相等时，错误信息会给出 v.x 的值， 方便定位问题。而 assertTrue() 只会提示 False != True。

运行测试#

使用命令 python -m unittest 运行测试。

运行指定的测试脚本：python -m unittest test.test_vector.TestVector.test_add
